### 题目
1.  已知二次函数的图象经过点 $(0, -3)$、$(1, 0)$、$(-1, -4)$，求这个二次函数的解析式。

### 正确答案
1.  $y = x^2 + 2x - 3$

### 解题步骤
1.   核心解题思路梳理：已知二次函数图象经过三个点，通常可以设二次函数的一般式为 $y = ax^2 + bx + c$，将三个点的坐标代入，得到一个三元一次方程组，解这个方程组即可求出 $a$、$b$、$c$ 的值，从而得到二次函数的解析式。
2.   分步推导过程：
     - 设二次函数的一般式为 $y = ax^2 + bx + c$。
     - 将点 $(0, -3)$ 代入，得到 $c = -3$。
     - 将点 $(1, 0)$ 代入，得到 $a + b + c = 0$，由于 $c = -3$，所以 $a + b - 3 = 0$，即 $a + b = 3$。
     - 将点 $(-1, -4)$ 代入，得到 $a - b + c = -4$，由于 $c = -3$，所以 $a - b - 3 = -4$，即 $a - b = -1$。
     - 解方程组 $\begin{cases} a + b = 3 \\ a - b = -1 \end{cases}$，将两式相加，得到 $2a = 2$，即 $a = 1$。
     - 将 $a = 1$ 代入 $a + b = 3$，得到 $b = 2$。
     - 因此，二次函数的解析式为 $y = x^2 + 2x - 3$。
3.  最终结论推导：二次函数的解析式为 $y = x^2 + 2x - 3$。
